# ProvEval: A Reproducibility Protocol and Unified Scoring System for Provenance-Based Security Tools

A standardized evaluation protocol and unified scoring metric (IOUS) for provenance-based APT detection systems.

## Paper

**Title:** Making APT Detection Research Reproducible: A Standardized Evaluation Protocol and Unified Scoring System for Provenance-Based Security Tools

**Author:** Maryam Shahzad

**PDF:** [ProvEval_APT_Detection_Reproducibility.pdf](ProvEval_APT_Detection_Reproducibility.pdf)

## Abstract

Provenance graphs have emerged as the dominant substrate for detecting Advanced Persistent Threats (APTs), but the field suffers from three critical gaps: irreproducible benchmarks, fragmented evaluation, and absence of open-source reference implementations. ProvEval addresses all three through a reproducibility protocol, the IOUS metric, and an open-source Python framework.
